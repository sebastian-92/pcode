# pcode
## a laser gcode parser in penguinmod
## [click to run online](https://pcode.cbass92.org)
## Info
### Supported codes:
#### Full support
 - M106
 - M107
 - G0
 - G1
 - X
 - Y
#### Partial support
 - R
 - I
 - J
 - G2
 - G3
### Usage
To use, right click the list and import a gcode file of your choice, then press space to render
